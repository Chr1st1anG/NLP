NLP Projekt - Heise Online Journal 

Das Projekt ist in drei Notebooks aufgeteilt, im Notebook "Download" befindet sich der Code welcher für das Herunterladen der Daten verwendet wurde. 
Im Notebook "Exploration" befindet sich die Projektbeschreibung und die erste Analyse der Daten, es wird empfohlen mit diesem Notebook zu Beginnen. 
Der Teil der Clusteranalyse und die Beschreibung der Ergebnisse befinden sich im Notebook "Clustering". 

Hinweise zur Ausführung: 
- Alle benötigten Bibliotheken sind in der Datei requirement.txt angegeben (Ausführung durch: pip install - r requirements.txt)
- Die Notebooks sind bereits ausgeführt, es kann jedoch sein, dass Grafiken, der Bibliothek plotly, teilweise nicht angezeigt werden. Diese müssen erneut ausgeführt werden.
- Um die Ausführung der Notebooks zu verkürzen, wurden Codezeilen, deren Ausführungszeit mehrere Stunden betrug, als pkl-Datei abgespeichert und auskommentiert.
  Die Ergebnisse werden bei Ausführung des Notebooks nun automatisch eingelesen.
